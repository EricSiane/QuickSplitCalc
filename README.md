# Quick Split Calc
A simple little plugin that adds a Split Option to your stack of coins and will tell you how much each person gets of a split

![splitcoins.png](images/splitcoins.png)
## Features
- Simple and easy-to-use interface
- Custom team size option
- Split coins evenly among a specified number of people![stanardsplit.png](images/stanardsplit.png)
- Rounding mode
